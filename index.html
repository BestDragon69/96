Règles du 96

Règles de base

  • Jeu avec 54 cartes (jokers inclus)
  • Chaque joueur reçoit 4 cartes
  • À chaque tour : poser une carte, annoncer le total, repiocher
  • But : ne pas dépasser 96 !

Cartes spéciales

  • As : vaut 1 ou 11 (au choix)
  • Valet : vaut 0
  • Dame : vaut -10
  • Roi : ramène le total à 70
  • Joker : vaut 0 et échange la main avec un autre joueur

Règles de boisson

  • Dizaines (10, 20, 30...) : distribuer N gorgées (1 pour 10, 2 pour 20...)
  • Multiples de 11 (11, 22, 33...) : tous les joueurs boivent N gorgées

Mise à mort

Si vous dépassez 96, vous devez réussir 4 étapes consécutives :

  • Étape 1 : Rouge ou Noir
  • Étape 2 : Plus ou Moins (que la 1ère carte)
  • Étape 3 : Interne ou Externe (entre les 2 premières)
  • Étape 4 : Couleur (♥♦♣♠)

En cas d'échec : retirer les cartes, boire autant de gorgées que l'étape, et recommencer depuis le début !
